# 🧠 AIML From-Scratch Project (3-Week Intensive)

A hands-on implementation of core Neural Network concepts **from scratch using Python**, focusing on understanding the fundamentals behind modern AI systems without relying heavily on high-level frameworks.

---

## 🚀 Projects Included

### 1. Optimizer Comparison
- Implemented XOR neural network using:
  - Vanilla Gradient Descent
  - Momentum
  - RMSProp
  - Adam
- Compared convergence speed and performance across optimizers

---

### 2. Custom Activation Function
- Designed a custom activation: **Sinu-ReLU**
- Compared its behavior and performance with standard ReLU
- Analyzed impact on learning dynamics

---

### 3. Regularization Techniques
- Addressed overfitting using:
  - L2 Regularization
  - Dropout
- Evaluated model generalization improvements

---

### 4. Deep Network Stability
- Built a 3-hidden-layer neural network
- Applied **He Initialization** for stable training
- Prevented vanishing/exploding gradients

---

### 5. Mini Autoencoder
- Implemented unsupervised learning model
- Used bottleneck architecture for feature compression
- Demonstrated reconstruction capability

---
